# sentence to graph
A Python wrapper to transform a sentence into a graph

# Quick start
```
$ git clone https://github.com/Himmelstein/snt2ldg
$ cd snt2ldg
snt2ldg $ make init
```

# Run test
```
snt2ldg $ make test
```

# Generate documentation
```
snt2ldg $ ./make_docu
```
Documentation is located at docs/html/
